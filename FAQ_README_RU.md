# Улучшения FAQ - Краткая документация

## Что сделано

Оптимизирована секция FAQ с папками для лучшей производительности на мобильных устройствах и исправлено наложение изображений на текст.

## Основные улучшения

### 1. Производительность
- Автоматическое определение мощности устройства
- Адаптивная скорость анимаций (200/300/400ms)
- GPU ускорение для плавности
- Оптимизация памяти на слабых устройствах

### 2. Исправлена проблема с изображениями и читаемостью текста
- Изображения больше не залезают на текст
- Правильное позиционирование при открытии папок
- Ограничены размеры изображений
- **Добавлен градиентный оверлей** для читаемости белого текста на цветных картинках
- **Text-shadow** для всего текста обеспечивает отличную читаемость
- **Backdrop-filter** (размытие фона) для современных браузеров

### 3. Мобильная оптимизация
- Отключены hover эффекты на мобильных
- Упрощенные анимации
- Лучшая производительность на слабых Android устройствах

## Тестирование

Проверьте на:
- ✓ iPhone Safari
- ✓ Android Chrome  
- ✓ Desktop (Chrome/Firefox/Safari)

Особое внимание на:
- Скорость открытия/закрытия папок
- Наложение изображений на текст
- Плавность анимаций

## Техническая информация

**Детектор производительности** автоматически определяет:
- Количество ядер CPU
- Объем RAM
- Скорость интернета
- Тип устройства (мобильное/desktop)

На основе этого выбирается оптимальная конфигурация анимаций.

**CSS переменная** `--faq-transition-duration` управляет скоростью всех анимаций и устанавливается автоматически.

## Поддержка

Код полностью совместим с:
- Современными браузерами (Chrome 90+, Firefox 88+, Safari 14+)
- iOS 14+
- Android 8+

Для старых браузеров анимации будут упрощены автоматически.
