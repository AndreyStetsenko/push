# Push Theme

Базовая тема WordPress для подключения готовой верстки.

## Структура

- `style.css` - основной файл стилей (заголовок темы)
- `functions.php` - функции темы, подключение стилей и скриптов
- `header.php` - шапка сайта
- `footer.php` - подвал сайта
- `index.php` - главный шаблон
- `page.php` - шаблон для страниц
- `single.php` - шаблон для записей
- `src/` - исходные файлы для сборки:
  - `scss/` - SCSS файлы
  - `js/` - JavaScript файлы
- `assets/` - скомпилированные файлы:
  - `css/` - скомпилированные CSS
  - `js/` - скомпилированные JS
  - `images/` - изображения

## Система сборки

### Установка

```bash
npm install
```

### Команды

- `npm run build` - сборка всех файлов
- `npm run watch` - отслеживание изменений и автоматическая сборка
- `npm run dev` - то же что и watch

### Работа с файлами

1. **SCSS**: Размещайте файлы в `src/scss/`, импортируйте через `@import`
2. **JS**: Размещайте файлы в `src/js/`, используйте ES6+ синтаксис
3. **PHP**: Верстка находится в PHP файлах темы (header.php, footer.php, index.php и т.д.)

Скомпилированные файлы автоматически попадают в `assets/` с source maps и минифицированными версиями.

## Подключение верстки

1. После сборки файлы будут в `assets/css/` и `assets/js/`
2. Раскомментируйте соответствующие строки в `functions.php` для подключения файлов
