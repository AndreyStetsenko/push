# Двойные изображения в FAQ - Техническая документация

## Проблема
При разном количестве контента на разных языках изображения "съезжали" в разные места, так как их позиционирование было привязано к высоте контента.

## Решение
Реализована система двойных изображений:
1. **Изображение в табе** - показывается когда папка закрыта
2. **Изображение в контенте** - показывается когда папка открыта

## Как это работает

### 1. PHP (faq.php)
Изображения выводятся в двух местах:

```php
<div class="folder__tab">
    <span class="title">Заголовок</span>
    <!-- Изображение для закрытого состояния -->
    <div class="folder__tab--image">
        <img src="..." alt="...">
    </div>
</div>

<div class="folder__content">
    <!-- Изображение для открытого состояния -->
    <div class="folder__content--image">
        <img src="..." alt="...">
    </div>
</div>
```

### 2. CSS анимация

**Закрытое состояние:**
```scss
.folder__tab--image {
    opacity: 1; // Видимо
}
.folder__content--image {
    opacity: 1; // Тоже видимо, но за пределами видимой области
}
```

**Открытое состояние:**
```scss
.folder.is-open {
    .folder__tab--image {
        opacity: 0; // Плавно исчезает
    }
    .folder__content--image {
        // Появляется снизу справа
    }
}
```

## Позиционирование

### Изображение в табе
- Находится рядом с текстом заголовка
- Размер: 60px высота на десктопе, 30px на мобильных
- Для соцсетей: горизонтальный ряд иконок
- `gap: 20px` между текстом и изображением

### Изображение в контенте
- Позиция: `bottom: 0; left: 80%` (правый нижний угол)
- Размер: адаптивный, с ограничением `max-height`
- Под текстом с градиентным оверлеем

## Преимущества

✅ **Независимость от контента**
- Изображение в табе всегда на одном месте
- Не зависит от количества текста в вопросах

✅ **Мультиязычность**
- Работает одинаково на всех языках
- Не требует настройки для каждого языка

✅ **Плавная анимация**
- Изображение в табе исчезает при открытии
- Изображение в контенте появляется
- Длительность: `var(--transition-time)` (200-400ms)

✅ **Адаптивность**
- Разные размеры для десктопа и мобильных
- Оптимизировано для touch-устройств

## Технические детали

### Transition
```scss
transition: opacity var(--transition-time) ease;
```
- Время зависит от производительности устройства
- 200ms на слабых, 400ms на мощных

### Pointer events
```scss
pointer-events: none;
```
- Изображения не перехватывают клики
- Клик всегда работает на табе/контенте

### Z-index
- Tab image: в контексте таба
- Content image: `z-index: 1` (под текстом)
- Text wrapper: `z-index: 2` (поверх изображений)

## Поддержка соцсетей

Для папки с соцсетями (Facebook, TikTok, Instagram):
```scss
.socials {
    display: flex;
    gap: 8px; // Расстояние между иконками
}
```

Каждая иконка:
- Desktop: 50px высота
- Mobile: 25px высота

- ✅ Все современные браузеры
- ✅ iOS 12+
- ✅ Android 8+
- ✅ IE11 (без backdrop-filter, но работает)
